## 📊 项目统计

### 文件统计
- 总文件数: **14个**
- 总代码行数: **6,500+ 行**
- JavaScript: 4,380 行
- HTML/CSS: 1,049 行
- 文档: 1,071 行

### 技术栈
- **前端框架**: Vue 3.4 (CDN)
- **渲染引擎**: HTML5 Canvas (高性能)
- **后端**: Firebase 9.22
  - Authentication (认证)
  - Firestore (数据库)
  - Cloud Storage (文件存储)
- **样式**: Tailwind CSS 3 + 自定义 CSS
- **图标**: Font Awesome 6.5
- **字体**: Inter, Kalam, JetBrains Mono
